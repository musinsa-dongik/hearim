# 헤아림 위클리 템플릿

아래 형식을 따라 작성한다.

```markdown
# [N]차: YYYY-MM-DD (요일) - [N]주차

**[N]차 YYYY-MM-DD (요일) - 스터디 3줄 요약**

1. [핵심 요약 1]
2. [핵심 요약 2]
3. [핵심 요약 3]

---

## 발표 1: [발표자 이름]

### 주제: [주제명]

**핵심 개념:**
- [개념 1]: [설명]
- [개념 2]: [설명]
- [개념 3]: [설명]

**코드/링크:**
- [관련 PR, 블로그, 문서 링크]

**설명이 막혔던 부분:**
- [막힌 지점과 이유]

**도움된 질문:**
- Q: [질문 내용]
- A: [답변/토론 내용]

**다시 공부할 지점:**
- [ ] [학습 포인트 1]
- [ ] [학습 포인트 2]

---

## 발표 2: [발표자 이름]

[동일 구조로 작성]

---

## [N]차 총평

[튜터의 총평]

- **두 발표의 연결고리**: [공통 주제나 시너지]
- **상호 학습 포인트**: [서로의 발표에서 배울 점]
- **추천 질문 (Action Item)**: [다음 주까지 탐구할 주제]
```

## 작성 체크리스트

**발표 전:** 주제 선정, 코드/링크 준비, 핵심 개념 3-5개 정리
**발표 중:** 10분 내외 설명, 막히는 부분 메모
**발표 후:** 핵심 정리, 도움된 질문 기록, 재학습 지점 명시
